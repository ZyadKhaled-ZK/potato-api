How It Works

The widget you see is injected by a one-line <script> tag added to this page. It fetches its configuration from the widget platform API, renders itself, and submits data back to the platform — all cross-origin.

Cross-Origin

This page runs on a different origin than the widget platform API. CORS headers allow the script to fetch config and submit data.

Cached Config

Widget configuration is served with cache headers so repeated loads are fast. The config payload is minimal.

Spam Protected

Rate limiting per IP/widget and honeypot fields prevent abuse without CAPTCHAs.

Geo Enrichment

Submissions are automatically enriched with IP-based geo data using a provider fallback chain.

Embed Snippet

To add this widget to any site, just paste this one line before </body>:

<script src="http://localhost:3000/widget.js" data-widget-id="1" async></script>
